## セットアップ

### 前提条件

- **macOS** 12.0 (Monterey) 以上
- **Node.js** 20.x 以上
- **npm** 9.x 以上

### インストール & 起動

```bash
cd spiral-v2
npm install
npm run dev        # 開発サーバー起動
npm run start      # 別ターミナルでElectron起動
```

### ビルド & パッケージング

```bash
npm run pack
# → release/Spiral-1.0.0.dmg が生成される
```

## 機能一覧

### タブ管理

**タブの作成・操作**
- `⌘T` で新しいタブを即座に開く（UIは即時反映、高速）
- `⌘W` でアクティブなタブを閉じる
- `⌘1` 〜 `⌘9` でタブを番号指定で切り替え
- タブは起動時に自動復元される（プライベートタブは除く）

**タブ名の編集**
- タブを**ダブルクリック**してタブ名をその場で編集できる
- 右クリックのコンテキストメニューから「タブ名を編集」も可能
- `Enter` で確定、`Escape` でキャンセル

**ピン留めタブ**
- 右クリック → Pin Tab でタブをサイドバー上部にアイコン表示で固定
- ピン留めタブはワークスペースをまたいで表示される

**タブグループ**
- 関連するタブをグループにまとめて整理できる
- グループごとに色と名前を設定可能

---

### Split View（画面分割）

**ドラッグで起動**
- サイドバーのタブを**ブラウザエリアにドラッグ＆ドロップ**するだけで左右分割が起動する

**手動起動**
- ツールバー右端の分割ボタンで起動
- アクティブタブと他のタブを左右に並べて表示

**分割比率の変更**
- 中央の仕切りをドラッグして自由に比率を変更できる（20%〜80%）
- 左右どちらのペインもクリックでフォーカスを切り替え、URLバーが連動

---

### Little Arc（ミニウィンドウ）

- `⌘⇧N` またはサイドバー下部の 🌀 ボタンで起動
- メインウィンドウの上に浮かぶ小型ブラウザ（600×420px）
- タイトルバーをドラッグして自由に移動できる
- 戻る・進む・リロード・ピン留め機能つき
- 緑のボタンでメインウィンドウに展開
- 複数のLittle Arcを同時に開ける

---

### スクリーンショット

- `⌘⇧4` またはサイドバー下部のカメラボタンで起動
- 現在のタブをフルページキャプチャ

**注釈ツール**

| ツール | 機能 |
|--------|------|
| 選択 | 範囲を指定してトリミング |
| 矢印 | ドラッグで矢印を描く |
| 四角 | 四角形を描く |
| 円 | 楕円を描く |
| ペン | フリーハンドで描く |
| テキスト | テキストを追加 |

**カラーパレット**: 赤・オレンジ・黄・緑・青・紫・白・黒の8色から選択

**エクスポート**
- 「保存」ボタンで PNG ファイルとしてダウンロード
- 「コピー」ボタンでクリップボードに画像としてコピー
- 選択ツールで範囲指定した場合はその領域のみ出力

---

### リンクプレビュー

- リンクの上に**600ミリ秒ホバー**するとポップアップでページをプレビュー表示
- プレビュー内の「開く」ボタンでそのまま遷移
- × ボタンまたは外側クリックで閉じる

---

### Picture in Picture（自動PIP）

- 動画が再生中にタブを切り替えると**自動でピクチャーインピクチャー**に移行する
- ページを非表示にしたとき（別アプリに切り替えなど）も自動でPIPが起動
- YouTube・Netflix・その他 `<video>` タグを使うサイトに対応

---

### ライブラリ

- `⌘⇧L` またはサイドバー下部のフォルダボタンで起動
- Finderを開かずにブラウザ内でファイルを管理・アップロードできる

**対応ファイル形式**

| 種別 | 内容 |
|------|------|
| 画像 | JPG, PNG, GIF, WebP など |
| 動画 | MP4, MOV, WebM など |
| 音楽 | MP3, AAC, FLAC など |
| 書類 | PDF, Word, テキストなど |
| その他 | 任意のファイル |

**操作**
- ドラッグ＆ドロップまたは「アップロード」ボタンでファイルを追加
- グリッド表示 / リスト表示を切り替え可能
- 名前・種別でフィルタリング・検索
- ファイルを選択してコピー・削除

---

### ワークスペース（Spaces）

- Arc の Spaces と同等の機能
- 複数のワークスペースを作成してタブを整理
- ワークスペースごとに名前・カラーを設定できる
- サイドバー上部のドロップダウンから切り替え
- ワークスペースを切り替えるとサイドバーのグラデーションカラーも変わる

---

### コマンドパレット

- `⌘K` で起動
- タブ切り替え・URL入力・ブックマーク検索・設定へのアクセスを一箇所で
- キーボードだけで完結する操作フロー

---

### ブックマーク

- URLバー右端の星アイコン、または `⌘D` でブックマーク追加・削除
- サイドバーのブックマークタブで一覧・検索・削除
- ブックマークはセッションをまたいで永続保存される

---

### 閲覧履歴

- サイドバーの履歴タブで閲覧履歴を時系列表示
- 検索・フィルタリング対応
- 個別削除・全削除が可能

---

### ダウンロード管理

- ダウンロード中はサイドバーのバッジに件数を表示
- 進捗バー・完了通知・ファイルを開くボタンつき
- ダウンロードキャンセル・保存先をFinderで開く機能

---

### プライベートタブ

- `⌘⇧N` でプライベートタブを開く
- 履歴・Cookie・キャッシュがセッション終了時にクリアされる
- タブ一覧でロックアイコンが表示される
- タブの永続化（起動時の復元）対象外

---

### パスワードマネージャー

- ログインフォームの送信を検知してパスワードの保存を提案
- 保存済みパスワードはサイドバーの設定 → パスワードタブで管理
- 個別削除対応

---

### DeepL翻訳（バンドル拡張機能）

- アプリ起動時に自動でインストールされる拡張機能
- テキストを選択して右クリック → 「DeepLで翻訳」でポップアップ翻訳
- ツールバーのアイコンから手動でテキスト入力して翻訳も可能
- 翻訳結果のコピー・DeepLウェブで開くボタンつき
- DeepL APIキー（無料プラン `:fx` 形式に対応）を設定すると高品質翻訳が使える
- APIキーなしでもDeepLウェブへの自動リンクとして機能

---

### 広告・トラッカーブロック

- 設定から有効/無効を切り替え可能
- 主要な広告配信ドメイン（DoubleClick、AdSense等）をブロック
- トラッカー（Google Analytics、Hotjar等）のブロックにも対応

---

### Reader Mode（リーダーモード）

- `⌘⇧R` で起動
- 広告や余分なUIを除去してテキストに集中できる表示に切り替わる

---

### Quick Note

- `⌘⇧O` でフローティングメモパッドを呼び出し
- タブに紐づいたメモとグローバルメモを切り替えて使用

---

### 開発者ツール

- `⌘⌥I` で Chromium デベロッパーツールを開く

## テーマ・カスタマイズ

### 組み込みテーマ

設定パネルの「外観」タブから切り替えられる。

| テーマ | 説明 |
|--------|------|
| **ライト** | 紫グラデーションサイドバー + ホワイトブラウザ（デフォルト） |
| **ダーク** | 深紫グラデーション + ダークブラウザ |
| **Midnight** | ブルー系ダークテーマ |
| **Rosé** | ピンク系ライトテーマ |

### Space カラー

設定パネルの「Space カラー」セクションから、サイドバーのグラデーションのアクセントカラーを変更できる。

- 10色のプリセットから選択
- カスタムカラーピッカーで任意の色を指定
- ワークスペース切り替え時に自動でカラーが変わる

### CSS カスタマイズ

`src/renderer/styles/themes.css` に CSS カスタムプロパティを追加することで独自テーマを定義できる。

```css
.theme-custom {
  --space-color: #FF6B6B;
  --sidebar-bg: linear-gradient(170deg, #2d1b1b 0%, #1a0f0f 100%);
  --accent-primary: #FF6B6B;
}
```

## アーキテクチャ

### プロセス構成

```
┌─────────────────────────────────────────────┐
│  Main Process (Node.js)                     │
│  WindowManager  TabManager  DownloadManager │
│  ExtensionManager  BlocklistManager         │
└──────────────┬──────────────────────────────┘
               │ contextBridge (preload.ts)
┌──────────────▼──────────────────────────────┐
│  Renderer Process (React + Vite)            │
│  browserStore.ts (Zustand)                  │
│  tabs / workspaces / bookmarks / history    │
│                                             │
│  <webview data-tabid="xxx">  × N tabs       │
└─────────────────────────────────────────────┘
```

### ナビゲーションのしくみ

戻る・進む・リロードは Renderer の `<webview>` に対して直接呼び出す。  
`webview.canGoBack()` / `webview.goBack()` を使うことで Main プロセスを経由せず即座に反映される。

### データ永続化

IPC ハンドラ `data:getTabs` / `data:setTabs` でタブの状態を保存。  
アプリ終了時にタブ一覧を保存し、次回起動時に自動復元（プライベートタブを除く）。

## ビルド・パッケージング

```bash
# レンダラー + メインプロセスを一括ビルド
npm run build

# macOS 向け DMG を生成（Intel + Apple Silicon）
npm run pack
# → release/Spiral-1.0.0-arm64.dmg
# → release/Spiral-1.0.0-x64.dmg
```

### 必要な macOS 権限

| 権限 | 用途 |
|------|------|
| `NSCameraUsageDescription` | Web 会議サイト |
| `NSMicrophoneUsageDescription` | Web 会議サイト |
| `NSLocationWhenInUseUsageDescription` | 位置情報サービス |
